# geometric_lib
Данный проект содержит функции, позволяющие получать результаты выражений стандартных значений для разных фигур.

## circle
### area
    Возвращает радиус круга.

    Параметры:
        r (int): радиус круга

### perimeter
    Возвращает периметр круга.

    Параметры:
        r (int): радиус круга

## Math formulas
### Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

### Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
